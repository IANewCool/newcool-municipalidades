# NewCool Municipalidades - Gobierno Local Chile

Portal educativo sobre municipalidades y tramites municipales en Chile.

## Demo

[https://municipalidades.newcool.io](https://municipalidades.newcool.io)

## Caracteristicas

- Buscador de municipalidades
- Guia de tramites municipales
- Calculadora de patente comercial
- Permisos y certificados
- Contacto con alcaldia

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
